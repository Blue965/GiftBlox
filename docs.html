Documentation

Guide complet pour utiliser GiftBlox

Démarrage Rapide

1

Ajouter le bot à votre serveur

Cliquez sur le bouton "Ajouter à votre serveur" sur la page d'accueil et suivez les instructions Discord.

2

Configurer le salon staff

Créez un salon privé pour le staff et ajoutez son ID dans le fichier .env du bot.

3

Ajouter des cadeaux

Utilisez la commande /ajouter-code pour ajouter des cadeaux à la boutique.

4

Créer des tâches

Utilisez la commande /ajouter-tache pour créer des tâches que vos membres peuvent compléter.

Commandes

/profil

Affiche votre profil avec vos points, niveau et XP.

/profil [utilisateur]

/boutique

Affiche la boutique avec les cadeaux disponibles.

/boutique

/acheter-carte

Achète une carte cadeau avec vos points.

/acheter-carte [type] [valeur]

/claim

Demande un paiement Robux avec vos points.

/claim [type] [valeur] [pseudo_roblox]

/taches

Affiche les tâches disponibles.

/taches

/completer-tache

Soumet une tâche comme complétée.

/completer-tache [id_tache] [preuve]

FAQ

Comment fonctionne le système de paiement ?

Notre système utilise une validation manuelle "Click-to-Approve". Quand vous demandez un paiement, vos points sont déduits immédiatement (anti-fraude), puis la demande est envoyée au staff pour validation. Le staff valide manuellement le paiement sur Roblox et vous notifie quand c'est fait.

Est-ce que le bot génère automatiquement des Robux ?

Non. GiftBlox est un système de fidélité, pas un générateur de Robux. Vous devez avoir vos propres Robux à distribuer. Le bot facilite seulement la gestion et la distribution sécurisée.

Le bot est-il sécurisé ?

Absolument. Le bot n'utilise aucun cookie ou mot de passe Roblox. Tous les paiements sont validés manuellement par le staff via le système officiel de groupes Roblox. C'est 100% légal et sécurisé.

Comment ajouter des cadeaux à la boutique ?

Utilisez la commande /ajouter-code avec le type de cadeau, la valeur, le code et le coût en points. Seuls les utilisateurs avec les permissions appropriées peuvent utiliser cette commande.

Que faire si je n'ai pas assez de points ?

Vous pouvez gagner des points en envoyant des messages dans le serveur (1 point par message avec cooldown de 60 secondes) et en complétant des tâches disponibles via /taches.

Puis-je utiliser le bot sur plusieurs serveurs ?

Oui, GiftBlox peut être ajouté à plusieurs serveurs Discord. Chaque serveur a sa propre base de données et ses propres paramètres.
