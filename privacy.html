Politique de Confidentialité

Dernière mise à jour : Juin 2026

1. Introduction

Chez GiftBlox, nous prenons votre confidentialité très au sérieux. Cette Politique de Confidentialité explique comment nous collectons, utilisons et protégeons vos données personnelles lorsque vous utilisez notre bot Discord et nos services.

2. Données Collectées

Nous collectons les types de données suivants :

  • Informations Discord : ID utilisateur, nom d'utilisateur, avatar, discriminateur
  • Données de compte : Points, XP, niveau, historique des tâches
  • Données d'utilisation : Fréquence d'utilisation, fonctionnalités utilisées
  • Préférences : Paramètres de langue, thème, notifications

3. Utilisation des Données

Nous utilisons vos données pour :

  • Fournir et améliorer nos services
  • Gérer votre compte et vos points
  • Personnaliser votre expérience
  • Communiquer avec vous (si vous avez activé les notifications)
  • Analyser l'utilisation pour améliorer le bot
  • Prévenir les abus et fraudes

4. Stockage des Données

Vos données sont stockées de manière sécurisée :

  • Dans une base de données SQLite sécurisée
  • Sur des serveurs situés dans l'Union Européenne
  • Encrypted pour les données sensibles
  • Accessibles uniquement par le personnel autorisé

5. Partage des Données

Nous ne vendons pas vos données personnelles. Nous pouvons partager vos données uniquement dans les cas suivants :

  • Avec votre consentement explicite
  • Pour se conformer aux obligations légales
  • Pour protéger nos droits et propriété
  • Avec des prestataires de services de confiance (hébergement, etc.)

6. Cookies et Stockage Local

Nous utilisons le stockage local du navigateur pour :

  • Sauvegarder vos préférences (thème, langue)
  • Stocker votre token de connexion Discord
  • Mémoriser vos paramètres de notification

Ces données sont stockées localement sur votre appareil et ne sont pas partagées avec des tiers.

7. Vos Droits

Conformément au RGPD, vous avez les droits suivants :

  • Accès : Demander une copie de vos données
  • Rectification : Demander la correction de vos données
  • Suppression : Demander la suppression de vos données
  • Portabilité : Demander le transfert de vos données
  • Opposition : Vous opposer au traitement de vos données

Pour exercer ces droits, contactez-nous à support@giftblox.com

8. Sécurité

Nous mettons en œuvre des mesures de sécurité appropriées pour protéger vos données :

  • Chiffrement des données sensibles
  • Accès restreint aux données
  • Surveillance régulière des systèmes
  • Mises à jour de sécurité régulières

9. Protection des Mineurs

GiftBlox n'est pas destiné aux enfants de moins de 13 ans. Nous ne collectons pas sciemment des données personnelles d'enfants. Si nous découvrons que nous avons collecté des données d'un mineur sans consentement parental, nous prendrons les mesures nécessaires pour les supprimer.

10. Modifications

Nous pouvons modifier cette Politique de Confidentialité à tout moment. Les modifications seront publiées sur cette page avec la date de mise à jour. Nous vous encourageons à consulter régulièrement cette page pour rester informé.

11. Contact

Pour toute question concernant cette Politique de Confidentialité ou pour exercer vos droits, n'hésitez pas à nous contacter :

Email : support@giftblox.com
Discord : Rejoignez notre serveur de support
